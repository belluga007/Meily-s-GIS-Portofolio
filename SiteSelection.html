Site Selection Analysis
Short Description
a decision-making process using geographical information systems to evaluate, rank, and select the optimal location for a facility, project, or development. It integrates spatial datasets (land use, demographics, transportation, topography) to perform suitability modeling, filtering out, and ranking potential locations based on defined criteria.
This map shows the site selection analysis for a new hospital in Jakarta. The analysis considers various factors such as population density, proximity to existing healthcare facilities, accessibility via transportation networks, and land use patterns.

The resulting map highlights the most suitable locations for the new hospital based on these criteria.
The analysis is performed using ArcGIS Pro, which allows for the integration of multiple spatial datasets and the application of weighted overlay techniques to identify optimal sites for development.
The map also includes a heatmap layer that visualizes areas of high population density, which is a critical factor in determining the demand for healthcare services in different parts of the city.
Overall, this site selection analysis provides valuable insights for urban planners and decision-makers in identifying the best location for a new hospital to serve the needs of the community effectively.

ps : I have excluded the water body in the classified process because the water body is not suitable for building a hospital, so I focused on the land area for the analysis.
Tools and Data
Method : Overlay and weightned analysis (Multi-Criteria Decision Analysis) to rank potential sites
Tools : ArcGIS Pro
  1. Jakarta Population density data (Jakarta Central Bureau of Statistics)
  2. Proximity to major roads (Indonesia Geospatial Data, Jakarta Province)
  3. Slope data (Indonesia National DEM (DEMNAS), Indonesia Geospatial Data)
  4. Land cover data (ESRI, Sentinel-2 Land Cover Explorer)
  5. Proximity to other hospitals
