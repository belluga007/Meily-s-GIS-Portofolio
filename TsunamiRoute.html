Closest Facilities Analysis: Fastest Route For Tsunami Evacuation Plan
Short Description
A map depicting the fastest evacuation routes from tsunami hazard zones to temporary and permanent evacuation points. The fastest route is calculated based on road length in meters. Several impediments, such as elevation, road type, and travel time, are ignored due to attribute limitations in the road dataset used.
Some data obtained by digitizing the Tsunami Evacuation Plan Map For Bantul Regency, Yogyakarta, 2010 consists of: estimated tsunami zones, fishing settlements, temporary and permanent evacuation points.
Tools and Data
Method : Digitizing, Network Analysis
Tools : Digitation, ArcMap
  1. Indonesia Topographic Map, Bantul Regency, Yogyakarta Special Region, 2019, Indonesia Geospatial
  2. Road Shapefile, 2019
  3. Tsunami Evacuation Plan Map For Bantul Regency, Yogyakarta, 2010
Deficiencies and possible improvements
  1. I tried to assign elevation attributes to the road dataset but it failed and I haven't found the reason yet.
  2. Determining the fastest route based on the length of the road in meters. I tried using other parameters such as "walking" but the network did not work and an error occurred.
  3. Due to the limited data on the road dataset attributes that I have, as a disaster response plan analysis, it would be better if vehicle speed, elevation, and time estimates were included in the analysis.
  4. Road type classification is ignored. The fastest route is measured without considering whether the road is passable by vehicles.
  5. There is an evacuation point that is at a steep height, if elevation is included in the "cost" it will probably give different results.
