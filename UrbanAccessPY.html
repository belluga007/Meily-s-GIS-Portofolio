ANALYZING URBAN ACCESS WITH PYTHON & OSM
Short Description
used OpenStreetMap, to match today’s topic, to pull both the road network and key POIs (schools, shops, clinics, gyms, etc), and then used the recently published #city2graph package to measure how connected things really are. The idea is simple: for a given point in a city, how much of your daily life can you reach within a walkable distance? The analysis are on: Zurich, Vienna, Budapest, and London.
Tools and Data
